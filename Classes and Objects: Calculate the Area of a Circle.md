4. **Execute the program**: Create an object of the class and call the method with the radius value.

## ðŸ§¾ Program
```
import math
class Saveetha:
    def __init__(self, r):
        self.area = math.pi * (r ** 2) 
    def slot(self):
        print("Area of circle:", round(self.area, 2))
r = float(input())
res = Saveetha(r)  
res.slot()
```
